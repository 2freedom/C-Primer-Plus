###第8章字符输入/输出和输入验证
---
如果用一个特殊字符（如，#）来结束输入，就无法在文本中使用这个字符，是否有更好的方法结束输入？
#### 无缓存（直接）输入
回显用户输入+立即重复打印该字符，即正在等待程序可立即使用输入的字符。
#### 缓存输入（缓冲区buffer中输入可修正，块传输）
Enter键之前不会重复打印刚输入的字符，Enter键之后程序才可使用输入字符。
- ##### 完全缓冲I/O	:buffer填满后刷新（内容被发送至目的地），文件输入
- ##### 行缓冲I/O	:出现换行符时刷新，键盘输入
#### conio.h
- getche()	回显无缓冲输入
- getch()	无回显无缓冲输入
